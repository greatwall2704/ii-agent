Conditional Flow Matching (CFM)

Path Construction

  • Sample endpoints: x 0 ∼p 0 , x 1 ∼p data .
  • Define an interpolant path x t |x 0 ,x 1 (linear or stochastic).
  • Compute conditional velocity u t (x t |x 0 ,x 1 ).

Training Loss

  • MSE: E[||v θ (x t ,t)−u t (x t |x 0 ,x 1 )||²].
  • Unbiased for the marginal velocity field.
  • Works for unconditional and conditional generation (e.g., text).
