Implementation Roadmap (12–18 Months)

Execute in phases; rebalance on data, not headlines.

Phase 1 (Month 0–1)

  • Open/verify brokerage and bank accounts; enable USD conversion.
  • Place deposit ladder; start VGB DCA.
  • Buy initial 30–40% of target gold/USD allocations.

Phase 2 (Month 2–6)

  • Build equity core via VN30 ETF in 3–5 tranches.
  • Add thematic exposures on dips (logistics/industrial parks).
  • Evaluate 1–2 IG corporate bonds.

Phase 3 (Month 7–18)

  • Review tariffs/legal updates quarterly; adjust USD/exports tilt.
  • Rebalance to bands; harvest gains; reinvest coupons.
  • Consider small real estate/REIT sleeve if financing improves.

Phased execution reduces timing risk.
